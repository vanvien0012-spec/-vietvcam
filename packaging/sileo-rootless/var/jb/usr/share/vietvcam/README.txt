VietVCam Control (Sileo rootless package)

This package installs a launcher command:
  /var/jb/usr/bin/vietvcam-control

Default control URL:
  http://127.0.0.1:8080/control

To change URL, edit:
  /var/jb/var/mobile/Library/Preferences/com.vanvi.vietvcam.conf

Example:
  VIETVCAM_CONTROL_URL=http://192.168.1.23:8080/control

This package does not bypass jailbreak detection and does not patch target apps.
